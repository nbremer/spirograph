Spirographs

On this webpage you can go crazy in creating spirographs!
Click anywhere to draw a random shape, either dashed or solid

Use the buttons in the control box to have more control. Draw a dashed spirograph, remove the last drawn shape or remove them all with a reset.

The set-up was originally created as an example for my D3 tutorial "Animated (dashed) lines in d3.js with Spirographs". This version has been expanded to offer you the freedom to set your own spirograph parameters.

In the "Parameters" folder of the control box you can see 4 different adjustable variables to make any shape that you like. Try to envision how the spirograph would look like in real life, a small circle rotating inside a larger fixed circle (see the image below).
There is the duration of the drawing (in seconds). The radius of the outer circle, in which the smaller circle (with its own radius) goes around. And finally, there is the location of the pen hole inside the smaller circle. The distance between the center of the small circle and the pen hole is given by ρ (rho). In this visual I’ve used the physical contstraints that ρ < r and r < R.

If you do not adjust the settings and draw a spirograph, either by clicking anywhere on the page or by using the draw buttons in the control box, a random set of values will be used for the 3 radii. So you can click away and get a new shape each time. However, once you change one of the three radii values, the next spirograph will use these values for its settings.

See the wiki page for info on the mathematical basis of Spirographs. If you'd like to save the resulting image as an SVG I'd suggest using SVG Crowbar. Very simple and extremely useful in many situations.
